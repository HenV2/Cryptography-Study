# Cryptography-Study

A collection of projects and implementations covering cryptography from basic to modern methods.

Includes :

- Ceaser Cipher
- Diffie Hellman
- Elgamal
- Digital Signatures
- euler phi function
- elliptic curves
- Lentsra's factorization
- Pollard's p1
- RSA
- Goldwasser-Micali

Proofs are not included, but I did study them as well.
